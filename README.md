# Крестики-Нолики

Крестики-Нолики — это браузерная версия классической игры Tic-Tac-Toe, доступная прямо в браузере без установки.

🌐 Website  
https://крестики-нолики.рф/

## About

Проект recreates the classic Tic-Tac-Toe game in a simple browser format.

The goal is to provide quick access to the game without downloads, registration, or complex setup.

Players can open the page and start playing instantly.

## Gameplay

Players take turns placing marks on a 3×3 grid:

- one player uses X
- the other uses O
- the goal is to place three symbols in a row

Rows can be horizontal, vertical, or diagonal.

## Features

- Classic Tic-Tac-Toe gameplay
- Simple browser-based interface
- Works on desktop and mobile devices
- Instant play without installation
- Quick casual game sessions

## Philosophy

Крестики-Нолики follows the idea of simple web games that can be opened and played instantly.

The project focuses on accessibility and quick gameplay.
